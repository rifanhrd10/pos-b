# Phase 2 + Phase 10 + Phase 3 + AI Sprint Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Menyelesaikan gap Phase 2 dan Phase 10, membangun Phase 3 Product & Category Management, lalu menambahkan AI recommendation + OCR menu import berbasis Gemini.

**Architecture:** Delivery dibagi menjadi 3 block: Block A menyelesaikan RBAC/Employee dan Outlet Management, Block B membangun Product Management core, Block C menambahkan layer AI di atas product form yang sudah stabil. Semua AI output diperlakukan sebagai draft suggestion dan wajib direview user sebelum disimpan ke database.

**Tech Stack:** Next.js 15 App Router, Prisma/PostgreSQL, NextAuth v5, Zod, Tailwind CSS, Gemini (`@google/generative-ai`), local file upload (`/public/uploads`)

## Global Constraints

- Semua server actions gunakan `"use server"` directive
- Validasi selalu lewat Zod sebelum menyentuh DB
- Error response selalu `{ error: string }`, success selalu `{ success: true }`
- String waktu format `"HH:mm"`
- AI output tidak boleh langsung write ke DB tanpa review user
- Jika Gemini gagal, recommendation harus fallback ke template built-in
- OCR upload max 5MB, format: jpg/png/webp
- Ikuti pola existing: `prisma.ts` singleton, `auth()` dari `@/lib/auth`
- Permission dan outlet-aware query harus graceful fallback, tidak crash kalau context null
- Commit setiap task selesai

```typescript
import { prisma } from "@/lib/prisma";

export function requirePermission(
  userPermissions: string[],
  required: Permission | Permission[],
  all = false
): boolean {
  const reqs = Array.isArray(required) ? required : [required];
  return all
    ? reqs.every((permission) => userPermissions.includes(permission))
    : reqs.some((permission) => userPermissions.includes(permission));
}

export async function getCurrentEmployeePermissions(userId: string, businessId: string): Promise<string[]> {
  const employee = await prisma.employee.findFirst({
    where: { userId, businessId, isActive: true },
    include: { role: true },
  });

  return (employee?.role?.permissions as string[] | null) ?? [];
}
```

- [ ] **Step 1: Create `src/lib/outlet-context.ts`**

```typescript
import { cookies } from "next/headers";

const ACTIVE_OUTLET_COOKIE = "active_outlet_id";

export async function getActiveOutletId(): Promise<string | null> {
  const store = await cookies();
  return store.get(ACTIVE_OUTLET_COOKIE)?.value ?? null;
}

export async function setActiveOutletCookie(outletId: string | null) {
  const store = await cookies();
  if (!outletId) {
    store.delete(ACTIVE_OUTLET_COOKIE);
    return;
  }
  store.set(ACTIVE_OUTLET_COOKIE, outletId, { path: "/", httpOnly: true, sameSite: "lax" });
}
```

- [ ] **Step 2: Add `switchActiveOutlet` to `src/actions/outlets.ts`**

```typescript
import { setActiveOutletCookie } from "@/lib/outlet-context";

export async function switchActiveOutlet(outletId: string | null) {
  await setActiveOutletCookie(outletId);
  return { success: true };
}
```
